# Mon Appli

## Pré-requis
   * [GIT] **dernière version**
   * [Maven] **3.6.3**
   
## Instructions
* Cloner le dépôt avec la commande dans le répertoire de votre choix :
  > git clone https://github.com/DaviJam/testMaven 
* Pour lancer l'application, double cliquer sur le fichier : 
   >  start.bat
